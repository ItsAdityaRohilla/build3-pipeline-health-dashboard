# BUILD3 Weekly Pipeline Health Dashboard

## Overview

This project is an interactive Power BI dashboard built to monitor the weekly health of a startup accelerator pipeline.

The dashboard tracks:

- Applications
- Pipeline Velocity vs SLA
- Selection Rate
- Commercial Outcome
- Stage Bottlenecks
- Founders Requiring Attention


## Tools Used

- Power BI
- Microsoft Excel
- DAX
- Business Intelligence



## Key Features

Weekly KPI Monitoring
Pipeline Velocity Analysis
Bottleneck Identification
Weekly Operational Insights
Interactive Filtering
